# 深潜构筑 Demo 项目边界说明

## 1. 项目目标
本项目的目标，是完成一个可用于作品集展示的 2D 俯视角水下 Roguelike 动作 Demo。
重点在于验证核心玩法闭环，而不是制作一个内容庞大的完整商业游戏。

## 2. 核心卖点
- 海洋生物特征构筑
- 深度推进
- 氧气压力

## 3. 本版本必须实现的内容
- 1 名玩家
- 基础移动、冲刺、攻击、受击、死亡
- 3–5 种普通怪
- 1 种精英怪
- 1 个 Boss
- 房间式推进
- 奖励三选一
- 2 套基础流派
- 1 局完整流程（开始 -> 战斗 -> 成长 -> Boss -> 结算）

## 4. 本版本明确不做的内容
- 不做开放大地图探索
- 不做经营系统
- 不做多角色切换
- 不做大型剧情演出
- 不做复杂局外养成
- 不做大量武器类型
- 不做多人联机
- 不做超大体量关卡内容

## 5. 开发原则
- 先做成立，再做丰富
- 先验证玩法，再补内容
- 先完成 P0，再考虑 P1 / P2
- 如果某个功能不能明显提升主循环体验，就暂时不做

## 6. 后续判断标准
如果一个新想法不能直接增强以下任意一点，就先不加入当前版本：
- 战斗体验
- 构筑选择
- 房间推进
- 深度与氧气压力
- Boss 战表现
## 输入方案
本项目使用 Unity 旧输入系统（Input Manager (Old)）。
后续玩家移动、攻击、冲刺全部基于旧 Input API 开发，不切换到 New Input System。
